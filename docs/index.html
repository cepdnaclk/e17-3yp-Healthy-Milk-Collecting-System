Introduction

In dairy industry, dairy collectors measure only few parameters when buying milk from dairy farmers. But, they usually don't measure quality parameters like fat content, acidity etc of milk at that spot. Also the financial deals are happening in a traditional way. The quality is very important in dairy related products. Currently small scale manufacturers do not use technology based methods to measure the quality of milk. They just use organoleptic tests such as smell and visual observation for that.

Our targeted system can be used to measure and keep the records of important quality parameters of milk when the deal happens between the dairy collector and farmer to ensure milk is healthy and also it can be used to calculate financial value according to those parameters and manage the financial system in a cloud based system.

Solution Architecture

Measurements

Using our hardware unit following measurements of milk can be taken.

  • 1.PH value
  • 2.Fat rate
  • 3.Volume
  • 4.Weight
  • 5.Temperature
Using these measurements as primary measuremrnts, Quality parameters which are focused on software sector are calculated.
  • 1.PH value
  • 2.Fat rate
  • 3.Corrected density
Density can be calculated using weight and volume at measured temperature. Then, that value is converted to the parameter 'corrected density' considering temperature deviation from room temperature. Simply, density is mapped for room temperature.

PH value
Fat rate

A high intensity LED is used as a light source. A LDR is placed to detect the amount of light passing through milk. The more the fat content in the milk, more will be the amount of light scattered by milk. The circuit is calibrated using standards with sample of known fat values

  • cow's milk fat should be in range of 3.5 to 4.5 percent
  • buffalo's it is 6-7 percent.
